Membuat form login

Hai teman-teman, gimana kabarnya? Semoga sehat ya. Pada blog ini aku mau membuat halaman form registrasi. Nah kalian pasti sudah sering banget nih nemu halaman website dengan tampilan awal form untuk registrasi. Biasanya terdapat kolom untuk nama/username dan password.

Form ini digunakan untuk mendaftarkan nama/Username yang nantinya untuk kalian login melalui form login. Langsung aja kita mulai, buat 2 file (HTML & CSS) lalu hubungkan file CSS pada HTML dan buat source code seperti di bawah

Nah kode diatas sudah membuat tampilan form register tersebut menjadi berwarna karena menggunakan CSS untuk merapihkan, memperindah tampilannya serta menjadi jelas dan mudah dibaca

Pada kode tersebut kita hanya menggunakan 2 kolom, yaitu kolom Email dan password. Dan apabila sudah jadi maka tampilan dari halaman web akan menjadi seperti dibawah ini.

Jika sudah kamu saling source codenya, maka tinggal kamu jalankan pada tampilan web, hingga muncul halaman seperti yang di atas.
